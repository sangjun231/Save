## 프로젝트 소개
* 해외 여행객을 대상으로 한국의 현지 투어 상품을 판매하고 현지인이 직접 가이드해주는 투어 플랫폼
 
* 이 사이트는 지루하고 흔한 여행이 아닌, 현지인만이 아는 여행 장소와 맛집 등 특별한 투어를 계획하여 판매합니다. 

* 여행객은 현지 가이드와 실시간 소통을 통해 여행 계획 수립에도 도움을 얻을 수 있습니다.
